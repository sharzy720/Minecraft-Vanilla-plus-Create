# Minecraft-Vanilla-plus-Create
This is a combination of 2 mod packs
- [Better MC](https://www.curseforge.com/minecraft/modpacks/better-mc-forge-bmc4)

- [Create: Ultimate Selection](https://www.curseforge.com/minecraft/modpacks/create-ultimate-selection)

"Better MC" is the base of this modpack. With Create and its quests taken from "Create: Ultimate Selection"
